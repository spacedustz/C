## 표준 입출력 함수

| 표준입력함수    | 기능              |
| --------- | --------------- |
| scanf()   | 여러 종료의 자료 입력 받음 |
| getchar() | 1개의 문자 입력 받음    |
| gets()    | 문자열 입력 받음       |

<br>

| 표준출력함수    | 기능           |
| --------- | ------------ |
| printf()  | 여러 종료의 자료 출력 |
| putchar() | 1개의 문자 출력    |
| puts()    | 문자열 출력       |
